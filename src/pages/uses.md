# What I Use

**Updated November 3, 2025**

## Development

**Editor**: [Zed](https://zed.dev) – A beautiful, fast IDE with AI built in. Not built on electron. Still learning neovim.

**Terminal**: [Ghostty](https://ghostty.org) with Tmux – GPU-accelerated, actually native. Tmux for sessions, windows, and panes.

**Browser**: Zen – Privacy-focused Firefox fork. A spiritual successor to Arc.

**Notes**: Bear – Clean markdown notes that sync through iCloud.

**Music**: [Brain.fm](https://my.brain.fm/payment?extended_promo=30&utm_source=referafriend) for focus sessions, Apple Music for everything else.

**RSS**: Newsboat – Terminal-based feed reader.
