# Book Title by Author Name

This is a template file for creating new book reviews. Replace this content with your actual review.

## How to Use This Template

1. Make a copy of this file with a descriptive filename (use kebab-case like `book-title-review.mdx`)
2. Update the frontmatter section (between the `---` markers) with the book details
3. Write your review using Markdown and MDX features
4. Set `draft: false` when ready to publish

## Frontmatter Fields

- **title**: The title of the book
- **description**: A brief summary of your review (good for SEO)
- **author**: The book's author
- **coverImage**: Path to the book cover image (put images in `public/covers/`)
- **rating**: Your rating on a scale of 1-5
- **pubDate**: Publication date of your review (YYYY-MM-DD format)
- **updatedDate**: Date of last update (optional)
- **readDate**: When you finished reading the book (optional)
- **tags**: Array of relevant tags (used for categorization)
- **genre**: Array of genres the book belongs to
- **isbn**: The book's ISBN (optional)
- **amazonLink**: Link to purchase the book (optional)
- **draft**: Set to `true` to keep it as a draft (only visible in development)

## Book Cover Images

For book covers:
1. Place image files in the `public/covers/` directory
2. Reference them with a path starting with `/covers/`
3. Example: `coverImage: "/covers/book-title.jpg"`

For external images, you can use a full URL: `coverImage: "https://example.com/image.jpg"`

## Review Structure

### Overview

Start with a brief overview of the book without spoilers. What's the general premise? What genre is it? What's the general feel?

### What I Liked

Discuss the aspects of the book that you enjoyed. This could include:
- The writing style
- Character development
- Plot pacing
- Themes explored
- Emotional impact

### What Could Be Better

If there were aspects of the book that didn't work as well, discuss them here. Try to be constructive.

### Who Should Read This

Recommendations for what kind of reader might enjoy this book.

### Favorite Quotes

Include some memorable quotes from the book:

> "This is an example quote from the book that resonated with me."

> "Another memorable passage that illustrates something important about the book."

## Rating Explanation

Explain why you gave the book the rating you did. What factors went into your decision?

## Publishing

When your review is ready:
1. Set `draft: false` in the frontmatter
2. Ensure the cover image is properly linked
3. Check for any spelling or grammatical errors
